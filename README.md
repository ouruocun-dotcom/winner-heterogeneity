# Winner Heterogeneity Controls Between-Replicate Variability in Microbial Competition

Reproducible code and data for all figures in:

> **Winner Heterogeneity Controls Between-Replicate Variability in Microbial Competition**
>
> Ruocun Ou, Nanyang Normal University
>
> Submitted to *Physical Review Letters*

## Core result

Between-replicate distance in competitive microbial communities is exactly predicted by:

$$d = \sqrt{2\,\mathcal{H}}$$

where $\mathcal{H} = 1 - \sum_i P_i^2$ is the **winner heterogeneity** (complement of the Herfindahl index of the outcome distribution). This parameter-free formula is validated on 102 microbial communities across two domains of life.

## Quickest way to reproduce (Google Colab)

All scripts run in [Google Colab](https://colab.research.google.com/) without any setup beyond `pip install openpyxl`.

1. Upload this repository to Colab or clone from GitHub.
2. Run `fig1_theory.py` (pure theory, < 1 second).
3. Run `fig2_validation.py` (processes experimental data, ~30 seconds).

```bash
pip install openpyxl  # needed for reading .xlsx files
python fig1_theory.py
python fig2_validation.py
```

### What `supplementary_analysis.py` verifies

1. **Eq. (2)**: Saddle-point formula is exact (max error < 10⁻¹³)
2. **Eq. (3)**: P₁ = logistic is an algebraic identity (max error < 10⁻¹⁶)
3. **Bacteria pairs**: n=49, Spearman r=0.73
4. **Bacteria trios**: n=48, Spearman r=0.73
5. **Yeast environments**: n=5, Spearman r=0.98
6. **Bradley-Terry prediction (FAILURE)**: r≈0.35, accuracy=52% (mentioned in Discussion)
7. **N=3 basin structure**: Symmetric trio gives H=2/3 as predicted
8. **Temporal dynamics**: Unanimous communities saturate, contested grow

- `fig1_theory.py` is purely analytical (no random seeds needed).
- `fig2_validation.py` processes published experimental data deterministically. Running on any machine produces identical output.
